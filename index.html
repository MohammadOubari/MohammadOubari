What is HTML



Hyprt Text Markup Language


HTML documents are described by ELEMENTS .

Main Element is the //< html >// element, to close it // < /html>// .

There are always tow diffrent keys of Elements in every structure and they are the "It's the Head and the Body", those are the main components of any web page.

THE HEAD ELEMENT

Can include a title for your web site, CSS Style, some information for the browser.

THE BODY ELEMENT

It's where all the visible stuff,contents such as : TEXT,LINKS,IMAGES,LIST's and more and more .......

DOCTYPE Declaration

It's not an HTML Element, it's an instruction to the browser about which version of HTML the page is written in.

HTML Element Reference

Basic HTML Tag Description



//< !DOCTYPE> Defines the document type. //


//< HTML> Defines an HTML document. //


//< HEAD> Contains metadata/information for the document. //


//< TITLE> Defines a title for the document. //


//< BODY> Defines the document's body. //


//< H1> to < H6> Defines HTML headings. //


//< P> Defines a paragraph. //


//< BR> Inserts a single line break. //


//< HR> Defines a thematic change in the content. //


//< !--...--> Defines a comment. //


//< STRONG> Defines Bold Font. //


//< EM> Defines Italic Font. //


//< U> Defines Underline for Text. //
